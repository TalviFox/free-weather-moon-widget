# Contributing to Talvi's Weather Widget (TWW)

First off, thank you for considering contributing to Talvi's Weather Widget! 

Before you dive in, please keep a few things in mind to set expectations:
- **Built for Personal Use:** I make these tools primarily for solving my everyday problems. As such, they are generally considered "feature complete."
- **Scope Creep:** You are absolutely welcome to suggest features, but please understand they may be denied if they fall outside the core scope of the widget.
- **Limited Time:** This is not my day job; I have a full-time job! Because of this, my open-source time is highly limited and generally prioritized strictly for bug fixes.
- **Do Your Homework:** Please don't just "submit to submit." If you are submitting a bug fix or tweak, ensure it's tested and you've provided clear context on what it fixes.

## How to Contribute

### 1. Report Bugs
If you find a bug, please open an issue describing the problem, how to reproduce it, and what you expected to happen. I prioritize bug fixes!

### 2. Suggest Enhancements
If you have an idea for a cool new feature, please open an issue to discuss it *before* writing code. This saves both of us time in case the feature gets denied for scope creep.

### 3. Submit Pull Requests
If you're fixing a bug (or submitting an approved feature), feel free to submit a pull request!
- Fork the repository.
- Create a new branch for your bugfix.
- Ensure your code doesn't break the existing UI, daylight calculation math, or fallback logic.
- Submit a Pull Request with a clear description of the issue and your changes.

## Development Setup
No build tools are required! This project uses pure Vanilla JS and CSS. Simply clone the repository and open `index-gov.html` or `index-open-meteo.html` directly in your browser to test your changes.
